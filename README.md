字符编码转换
============

binary 转化为 简单ASCII
--------------------------
* `dns_source`  为测试文件

  ``` bash
	perl binary_to_ascii.pl dns_source
  ```

binary 转化为 ASCII
-----------
  ``` bash
	perl binary_to_real_ascii.pl dns_source
  ```

binary 转化为 Latin-1
-----------
  ``` bash
	perl binary_to_latin-1.pl dns_source
  ```

binary 转化为 中文(部分)
-----------
* `test` 为测试文件

  ``` bash
	perl binary_to_utf8.pl test
  ```
